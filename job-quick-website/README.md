# Job Quick Website

This is a static website. You do not need to install code tools to preview it.

## Open on your computer

Open `index.html` in a browser.

## Publish the website

The easiest beginner option is Netlify:

1. Go to https://app.netlify.com/drop
2. Drag the whole `job-quick-website` folder into the page.
3. Netlify will give you a public website link.
4. Leads from the form will appear in your Netlify site's `Forms` section.

You can also upload the same folder to Vercel, Cloudflare Pages, or any normal static hosting.

## Main files

- `index.html`: page content
- `styles.css`: layout, responsive design, colors, fonts
- `script.js`: mobile menu, FAQ, form demo, appear animations
- `assets/`: images, logo, and icons
- `thank-you.html`: page shown after a successful form submission on Netlify
